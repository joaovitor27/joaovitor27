<h2 style="border-bottom: none; padding-bottom: 0;">Stack e ferramentas</h2>

Minha base principal é **`Python`** (com Django) e **`TypeScript`** (usando Node.js no backend e React + Vite no frontend). Na hora de criar interfaces, prefiro bibliotecas consolidadas como **`Material UI (MUI)`** ou **`Ant Design`** em vez de utilitários como Tailwind. Para que o software chegue ao mundo real com segurança e constância, gerencio a infraestrutura e o deploy com **`AWS`** (EC2, RDS, S3), **`Terraform`** e **`Docker`**.
